# CloudWatch_vnt
Client side for CloudWatch_vnt

## Getting Started
Open terminal/cmd

Run:

  `git init`

  `git clone https://github.com/galigev/CloudWatch_vnt.git`

## Installation

Navigate to the project folder:

   `cd ../CloudWatch_vnt`

Run:
	
   `npm install`

## Run Server

Run:

   `npm start`


The website will open on 'localhost:3001'
